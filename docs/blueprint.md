# **App Name**: Grade Journey

## Core Features:

- Grade Level Sections: Two main sections for 10th and 11th grade students, offering distinct content.
- Sequential Class Access: Sequential unlocking of classes within each period, requiring completion of the first class to access subsequent ones.
- School Period Division: Four school periods per grade, each containing six classes. Display them in order
- Period Quiz Unlock: A quiz at the end of each period is a pre-requisite to unlock the next period
- Certificate Generation: Display and generate completion certificate once all four periods have been finalized for a grade.

## Style Guidelines:

- Primary color: Saturated blue (#4285F4) for trust and learning. 
- Background color: Light blue (#E3F2FD), a desaturated shade of the primary color for a calm, educational environment.
- Accent color: Green (#34A853) to indicate progress and success, standing out against the blue palette.
- Body and headline font: 'Inter' sans-serif for readability.
- Use a clear, card-based layout to separate each class and period.
- Use progress animations after the completion of each quiz to create excitement and positive feedback for the student.
